### Бекенд рекламной системы

Это бекенд системы которая позволяет в реальном времени посылать
рекламу на рекламные площадки в магазинах (мониторы) и в других публичных
местах.

### Стек

- Java 8
- PostgreSQL
- DropWizard 1.1.0
- heroku
